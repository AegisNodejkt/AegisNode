## Reporting a Vulnerability

We take the security of AegisNode seriously. If you discover a security vulnerability, please do NOT open an issue. Instead, please follow these steps:

1.  **Email**: Send details of the vulnerability to <anggashela@gmail.com>.
2.  **Encryption**: If possible, encrypt your email using our PGP key (available upon request).
3.  **Details**: Include as much information as possible:
    *   Type of vulnerability (e.g., XSS, SQLi, RCE).
    *   Steps to reproduce.
    *   Affected versions.
    *   Proof of concept (if safe to share).

## Response Timeline

*   **Acknowledgment**: We will acknowledge your report within 48 hours.
*   **Assessment**: We will assess the severity and impact within 1 week.
*   **Fix**: We aim to release a fix for critical vulnerabilities within 2 weeks of assessment.
*   **Disclosure**: We will coordinate public disclosure with you after the fix has been released.

## Bounty Program

Currently, we do not have a formal bug bounty program. However, we publicly acknowledge and thank all security researchers who responsibly report vulnerabilities.
